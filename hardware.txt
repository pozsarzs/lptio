                               Interface of LPTIO
                               ~~~~~~~~~~~~~~~~~~~
I. Files
~~~~~~~~
  hardware.txt		this file
  lptio.pro		KiCAD project file
  lptio.sch		schematic draw file
  lptio.cmp		component list
  lptio.stf		component/module list
  lptio.net		netlist
  lptio.brd		printed circuit board file
  schema.ps		schematic draw (postscript)
  comp.ps		component side of this board (postscript)
  silk.ps		silk screen of this board (postscript)
  sold.ps		solder side of this board (postscript)

II. General information
~~~~~~~~~~~~~~~~~~~~~~~
  Ups: 5V DC
  Il:  220 mA
